# Expense Tracker (Terminal)

A **simple terminal-based expense tracker** built with Python. Allows users to track, save, and manage their expenses directly from the command line.

## Features

* Add, view, and delete expenses
* Stores expense data in a **local file** for persistence
* Categorizes expenses by type (e.g., Food, Transport, etc.)
* Shows totals and summaries for better budgeting
* Simple, easy-to-use terminal interface

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/LaytonL24/ExpenseTracker.git
```

2. Run the script:

```bash
python3 main.py
```

3. Follow the on-screen prompts to:

* Add a new expense
* View all expenses
* Search for expense
* Delete an expense
* Exit the program
